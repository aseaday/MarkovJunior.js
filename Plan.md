# The Working Plan of MarkovJunior.js

### MVP - June 12 2022
A minimum viable program that supports:
- Basic Nodes:
    - one
    - all
    - prl
    - markov
    - sequence
- render the results to canvas

There is no support for inference in the MVP


### Version 0.1 - June 31 2022

- Support All Nodes
- Support Inference
- Render 
  - Canvas
  - WebGPU
